Local, privacy-focused speech-to-text for Linux.
No cloud. No tracking. Just whisper.

v0.9.6

How It Works

Press a key. Speak. Your words appear.
One global hotkey triggers dictation. Speak naturally and your words are transcribed and pasted instantly. That's it.
Your voice never leaves your machine.
No cloud, no server, no one listening. All inference runs locally through whisper.cpp. Your audio stays on your hardware.
50+ languages. Switch mid-sentence.
Whisper auto-detects the language you're speaking. Switch between languages naturally — it keeps up.
One hotkey, any app. Text lands instantly.
System-wide evdev hotkey works in any application. Transcribed text is injected directly where your cursor is.

Features

System-Wide Hotkey Dictation
Press a global hotkey anywhere, speak, and text auto-pastes into your focused window. Works across all apps.
Audio File Transcription
Open WAV, MP3, OGG, M4A, OPUS, FLAC, or MP4 files. Get a timestamped transcript with searchable segments.
Export to TXT, SRT, VTT, CSV
Save transcripts as subtitles for video, plain text for notes, or CSV for data analysis.
50+ Languages
Switch languages mid-sentence. Whisper detects and transcribes seamlessly with zero configuration.
CUDA GPU Acceleration
Leverage your NVIDIA GPU for faster transcription. Falls back to CPU gracefully.
Privacy-Focused (Local Only)
All processing on your machine. No accounts, no telemetry, no cloud. Nothing leaves your computer.

Prerequisites

Ubuntu / Pop!_OS / Debian:

terminal 
# Build tools and GTK4/libadwaita development libraries
$ sudo apt install build-essential cmake libgtk-4-dev libadwaita-1-dev

# Audio capture (ALSA development headers)
$ sudo apt install libasound2-dev

# Wayland clipboard and text injection (for auto-paste)
$ sudo apt install wtype wl-clipboard

# Global hotkeys — add yourself to the input group (reboot required)
$ sudo usermod -aG input $USER

# Rust toolchain (if not already installed)
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Fedora:

terminal 
$ sudo dnf install gcc cmake gtk4-devel libadwaita-devel alsa-lib-devel wtype wl-clipboard
$ sudo usermod -aG input $USER

Arch Linux:

terminal 
$ sudo pacman -S base-devel cmake gtk4 libadwaita alsa-lib wtype wl-clipboard
$ sudo usermod -aG input $USER

Installation from Source

Clone the repository and build:

terminal 
$ git clone https://github.com/kwap/linux-whisper.git
$ cd linux-whisper
$ cargo build --release

Install system-wide:

terminal 
# Copy binary
$ sudo cp target/release/linux-whisper /usr/local/bin/

# Install desktop entry
$ APP=com.linuxwhisper.LinuxWhisper
$ sudo cp data/$APP.desktop /usr/share/applications/
$ sudo cp data/icons/hicolor/scalable/apps/$APP.svg \
  /usr/share/icons/hicolor/scalable/apps/

# Update icon cache
$ sudo gtk-update-icon-cache /usr/share/icons/hicolor/

Or use just for a one-liner:

terminal 
$ just release
$ just install

On first run, Linux Whisper will prompt you to download a Whisper model. You can change the model size in Preferences at any time.

Security Audit

Comprehensive audit across all six crates and the CI/CD pipeline. Clean and safe to distribute.

Zero Unsafe Code
No unsafe blocks in the entire codebase. All FFI goes through safe whisper-rs wrapper bindings.
No Data Exfiltration
No telemetry, analytics, or tracking. Only HTTPS model downloads from HuggingFace. Transcriptions never leave your machine.
SHA-256 Model Verification
Downloads verified against known SHA-256 hashes. Corrupt or tampered files rejected.
No Command Injection
External tools invoked with safe argument passing and stdin piping. No shell interpolation of user input.
Not a Keylogger
Evdev hotkey listener tracks only the configured combo. Does not capture, log, or transmit any other keystrokes.
Pinned Supply Chain
All CI/CD Actions and build tools pinned to commit hashes with integrity checks. No unpinned third-party code.

Covers: backdoors, data exfiltration, credential theft, file system abuse, privilege escalation, unsafe FFI, command injection, XSS, dependency supply chain, CI/CD pipeline.
