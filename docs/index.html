Local, privacy-focused speech-to-text for Linux.
No cloud. No tracking. Just whisper.

v0.9.0

How It Works

  • Press a key. Speak. Your words appear. That's it. That's the whole thing.
  • Your voice never leaves your machine. No cloud. No server. No one listening. Just you and your hardware.
  • It's fast. Whisper runs locally, right on your GPU — or your CPU. Either way, it just works.
  • Speak in English. Switch to Spanish mid-sentence. Throw in some French. It keeps up. Over 50 languages, zero configuration.
  • One hotkey, any app. Your text lands exactly where you need it — instantly, invisibly, effortlessly.

Features

System-Wide Hotkey Dictation
Press a global hotkey anywhere, speak, and text auto-pastes into your focused window. Works across all apps.
Audio File Transcription
Open any audio file (WAV, MP3, OGG, M4A, OPUS, FLAC, MP4) and get a timestamped transcript with searchable segments.
Export to TXT, SRT, VTT, CSV
Save your transcripts in the format you need. Subtitles for video, plain text for notes, CSV for data.
9 Whisper Model Sizes
From Tiny (74MB) to Large V3 (2.9GB). Trade speed for accuracy based on your hardware and needs.
50+ Language Support
Transcribe in over 50 languages. Switch languages mid-sentence — Whisper handles it automatically.
Native GTK4/libadwaita UI
Looks and feels like a proper GNOME app. Transcription window with search, copy, and export. No Electron.
System Tray with Pigeon Icon
A geometric low-poly pigeon lives in your tray. Left-click to toggle recording — its eye glows red. Right-click for menu. Stays out of your way.
CUDA GPU Acceleration
Leverage your NVIDIA GPU for faster transcription. Falls back to CPU gracefully when unavailable.
Privacy-Focused (Local Only)
All processing happens on your machine. No data leaves your computer. No accounts, no telemetry, no cloud.
Multilingual in Same Sentence
Switch between languages mid-speech without any configuration. Whisper detects and transcribes seamlessly.

Supported Platforms

Linux Whisper runs on modern Linux distributions with GTK4 and libadwaita. Tested on:

  • Pop!_OS 24.04 (primary development platform)
  • Ubuntu 24.04+ (and derivatives: Kubuntu, Xubuntu, etc.)
  • Fedora 40+
  • Arch Linux (rolling release)
  • Debian 13 (Trixie)+

Requires: GTK 4.10+, libadwaita 1.5+, PipeWire or PulseAudio, Wayland or X11. NVIDIA GPU with CUDA optional but recommended for large models.

Prerequisites

Ubuntu / Pop!_OS / Debian:

# Build tools and GTK4/libadwaita development libraries
$ sudo apt install build-essential cmake libgtk-4-dev libadwaita-1-dev

# Audio capture (ALSA development headers)
$ sudo apt install libasound2-dev

# Wayland clipboard and text injection (for auto-paste)
$ sudo apt install wtype wl-clipboard

# Global hotkeys — add yourself to the input group (reboot required)
$ sudo usermod -aG input $USER

# Rust toolchain (if not already installed)
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Fedora:

$ sudo dnf install gcc cmake gtk4-devel libadwaita-devel alsa-lib-devel wtype wl-clipboard
$ sudo usermod -aG input $USER

Arch Linux:

$ sudo pacman -S base-devel cmake gtk4 libadwaita alsa-lib wtype wl-clipboard
$ sudo usermod -aG input $USER

Installation from Source

Clone the repository and build:

$ git clone https://github.com/piotrkwapin/linux-whisper.git
$ cd linux-whisper
$ cargo build --release

Install the binary, desktop file, and icons system-wide:

# Copy binary
$ sudo cp target/release/linux-whisper /usr/local/bin/

# Install desktop entry and icons
$ sudo cp data/com.linuxwhisper.LinuxWhisper.desktop /usr/share/applications/
$ sudo cp data/icons/hicolor/scalable/apps/com.linuxwhisper.LinuxWhisper.svg /usr/share/icons/hicolor/scalable/apps/

# Update icon cache
$ sudo gtk-update-icon-cache /usr/share/icons/hicolor/

Or use just for a one-liner:

$ just release
$ just install

On first run, Linux Whisper will prompt you to download a Whisper model. You can change the model size in Preferences at any time.
